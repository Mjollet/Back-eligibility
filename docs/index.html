index

Module sams-eligibility implements rules engine which evaluates commercial and/or technical eligibility. It also exposes some utility functions to be used as API in other sams apps like sams-catalog

▸ contains(labelDef, input)

Helper function to construct evaluation string for "contains" operator. Based on target referenced by label's definition, if target's type is:

  • string - constructs a condition that checks if target is a subset of the input array
  • array - constructs a condition that checks if input is a subset of target array
